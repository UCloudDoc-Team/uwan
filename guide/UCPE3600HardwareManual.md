### CPE智能网关硬件基本信息

| 属性        | 规格                                                         |
| ----------- | :----------------------------------------------------------- |
| 产品        | UCPE3600                                                     |

| SIM卡插入口 | 一个内嵌式SIM卡插入口，支持Mini-Sim卡                        |
| LTE 模组    | 选配                                                         |
| Wi-Fi       | 选配                                                         |
| 天线        | 选配                                                         |
| 工作环境    | 温度0°C - 45°C，海拔高度小于5000m，相对湿度5% - 95%，非凝露  |
