### CPE智能网关硬件基本信息

## **型号与参数**

| 属性  | 配置详情                     |
| :---- | ---------------------------- |
| 尺寸  | 240mm * 155.5mm * 40mm              |
| WAN口 | 2 个光纤端口/ 2 个 RJ45 端口 |
| LAN口 | 4 个 RJ45 网口               |
| Wi-Fi | 无                           |
| USB   | 1 个 USB 2.0 接口            |
| 天线  | 无                           |
| 电源  | 一根电源线，DC 12V/3A        |
